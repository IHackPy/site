JTAG-Serial programmer

Based on the high-speed FT2232H chip, our JTAG-Serial programmer plugs into a Milkymist board and provides a smooth development experience:

  • Configuring the FPGA with the open source UrJTAG tool
  • Writing the flash
  • Providing a serial console for debugging the device
  • Using GDB for live debugging of software running on the board
